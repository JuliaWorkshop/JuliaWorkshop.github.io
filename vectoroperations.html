julia> a = [-1, 0, 1]
3-element Array{Int64,1}:
 -1
  0
  1

julia> length(a)
3

julia> sum(a)
0

julia> mean(a)
0.0

julia> std(a)
1.0

julia> var(a)
1.0

julia> maximum(a)
1

julia> minimum(a)
-1

julia> b = sort(a, rev=true)  # Returns new array, original not modified
3-element Array{Int64,1}:
  1
  0
 -1

julia> b === a  # === tests if arrays are identical (i.e share same memory)
false

julia> b = sort!(a, rev=true)  # Returns *modified original* array
3-element Array{Int64,1}:
  1
  0
 -1

julia> b === a
true
  • To add an item at the front, use the unshift!() function.
  • To insert an element into an array at a given index, use the splice!() function.
  • To add an item at the end of an array, use push!()
  • To remove the last item from an array, use pop!()
  • Once you know the indices, you can use deleteat!() to delete an element, given its index number. 

# Add stuff to the end of a list with push! and append!
push!(a,1)     # => [1]
push!(a,2)     # => [1,2]
push!(a,4)     # => [1,2,4]
push!(a,3)     # => [1,2,4,3]


# Arrays of a particular Type
b = Int8[4, 5, 6] # => 3-element Int8 Array: [4, 5, 6]

# Remove from the end with pop
pop!(b)        # => 6 and b is now [4,5]
