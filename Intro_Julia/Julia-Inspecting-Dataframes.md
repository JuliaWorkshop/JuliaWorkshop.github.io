RDatasets
==============================

* Julia has ***RDatasets.jl*** package providing easy way to use and experiment with most of the standard data sets which are available in the core of R. 
* To load and work with one of the datasets included in RDatasets packages, we need to install RDatasets as follows ???

* The RDatasets package provides an easy way for Julia users to experiment with most of the standard data sets that are available in the core of R as well as datasets included with many of R's most popular packages. 

==============================
<pre><code>
julia> using Pkg
julia> Pkg.add("RDatasets")
</code></pre>
